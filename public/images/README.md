# Cartella Immagini

Questa cartella contiene tutte le immagini e i loghi del sito BixPulito PRO.

## Struttura

### `/logos`
Inserisci qui i tuoi loghi:
- `logo.png` o `logo.svg` - Logo principale (consigliato: sfondo trasparente)
- `logo-white.png` o `logo-white.svg` - Versione bianca del logo (per sfondi scuri)
- `favicon.ico` - Favicon del sito (16x16, 32x32, 64x64 px)

### `/services`
Inserisci qui le immagini dei servizi (opzionale):
- `divani.jpg` - Immagine per la sezione divani
- `materassi.jpg` - Immagine per la sezione materassi
- `tendaggi.jpg` - Immagine per la sezione tendaggi
- `tappeti.jpg` - Immagine per la sezione tappeti
- `auto.jpg` - Immagine per la sezione interni auto

### `/hero`
Immagini per la sezione hero/banner principale:
- `hero-background.jpg` - Immagine di sfondo principale
- `chi-siamo.jpg` - Immagine per la pagina Chi Siamo

## Formati Consigliati

- **Loghi**: SVG (scalabile), PNG con sfondo trasparente
- **Foto**: JPG o WebP (ottimizzati per web, max 500KB)
- **Dimensioni**: 
  - Loghi: almeno 200x200 px
  - Foto servizi: 800x600 px o superiore
  - Hero images: 1920x1080 px

## Note

Dopo aver aggiunto i loghi, aggiorna i componenti per utilizzarli al posto delle icone placeholder.
